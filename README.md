# React + Vite TodoList

A **responsive Todo List** application built with **React** and **JavaScript**, featuring user authentication and password management.

## Features

- **User Authentication:** Register and log in with email and password.
- **Set Username:** Users can set a username after first login.
- **Todo Management:** Add, mark as done, and delete todos.
- **Password Reset:** 
  - **Local Reset:** Directly reset password in localStorage.
  - **Mock Email Flow:** Simulated email link to reset password.
- **Responsive Design:** Works on desktop and mobile devices.

## Tech Stack

- React (Functional Components & Hooks)
- JavaScript
- LocalStorage for data persistence
- CSS & Bootstrap for styling
